# Telecom-Customer-Churn-Dashboard
Telecom Customer Churn Dashboard using Python, Sql,  Power BI

##Tools used
-python(Numpy,Panda,Matplotlib,Seaborn)
-SQl
-Power BI

##Workflow
Collected raw telecom customer data from kaggle.
cleaned and transformed the data using python.
Imported cleaned data into PostgreSQL.
Perform SQl analysis.
Connected PostgreSQL to Power BI.
Bulit an interactive dashboard.

##Dashboard KPIs
-Total Employees
-Average Salary
-Average Age
-Attrition Rate
-Promotion Rate
